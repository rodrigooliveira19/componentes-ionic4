Instalando o ionic via node

Abra o prompt de comando no modo administrador e execute npm install -g ionic

Criando o projeto

Para cria o projeto é bem simples, vamos utilizar o comando abaixo

ionic start nomeProjeto tipo[tabs,black ...] --type=angular

Logo em seguida digite o seguinte comando no prompt cd nomeProjeto
